Работаем с fbo и fbs

Интегрируемся со всеми МП для правильной работы по FBS и учета остатков

FBO ( Fulfilment By Operator ) - это модель, в которой продажи товаров осуществляются непосредственно со склада маркетплейса

FBS ( Fulfilment By Seller ) - модель работы, при которой продавец осуществляет продажу продукции со своего склада
